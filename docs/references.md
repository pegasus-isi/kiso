# References

- [Pegasus Workflow Management System](https://pegasus.isi.edu) — Scientific workflow management system used by Kiso to define, execute, and monitor complex multi-step experiments.
- [EnOSlib](https://discovery.gitlabpages.inria.fr/enoslib/) — Infrastructure management library that Kiso builds on for provisioning resources across testbeds and managing remote execution.
- [Chameleon Cloud](https://www.chameleoncloud.org) — NSF-funded cloud testbed for computer science research; one of the primary testbeds supported by Kiso.
- [FABRIC](https://portal.fabric-testbed.net) — Nationwide research infrastructure providing programmable networking and computing; supported as a Kiso deployment target.
- [Vagrant](https://www.vagrantup.com/) — Tool for building and managing virtual machine environments; used by Kiso for local development and testing without a cloud account.
- [VirtualBox](https://www.virtualbox.org/) — Virtualization platform used as the default backend for Vagrant-based Kiso experiments.
