# Contact

How to reach the Kiso team and community.

## GitHub Issues

**Best for:** Bug reports, feature requests, and specific technical problems.

[https://github.com/pegasus-isi/kiso/issues](https://github.com/pegasus-isi/kiso/issues)

Open an issue for anything that can be tracked: a reproducible bug, a clear feature request, or a question with a definitive answer. Issues are public and searchable — check existing issues before opening a new one.

## GitHub Discussions

**Best for:** Release announcements and general questions.

[https://github.com/pegasus-isi/kiso/discussions](https://github.com/pegasus-isi/kiso/discussions)

New releases are announced on GitHub Discussions. Watch the repository to be notified of new releases and announcements.

## Email

**Best for:** Private questions, security disclosures, or enquiries not suitable for a public forum.

[kiso@isi.edu](mailto:kiso@isi.edu)

For security vulnerabilities, please use [GitHub's vulnerability reporting](https://github.com/pegasus-isi/kiso/security/advisories/new) rather than a public issue.
